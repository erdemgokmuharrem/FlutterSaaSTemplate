## 🚀 Kullanım Senaryosu (Müşteri)

### Adım 1: Satın Alma Sonrası (5 dakika)
```bash
git clone <repository>
cd FlutterSaaS
flutter pub get
```

### Adım 2: Firebase Setup (10 dakika)
```bash
# FIREBASE_SETUP.md takip et
# Config dosyalarını yerleştir
# Test et
```

### Adım 3: RevenueCat Setup (10 dakika)
```bash
# REVENUECAT_SETUP.md takip et
# API keys ekle
# Test et
```

### Adım 4: Özelleştirme (30 dakika)
```bash
# Branding değiştir
# Colors/theme güncelle
# App name değiştir
```

### Adım 5: Build & Deploy (Platforma göre)
```bash
# iOS: flutter build ios
# Android: flutter build apk
# Web: flutter build web
# macOS: flutter build macos
```

**Toplam Süre**: ~1 saat (setup + özelleştirme + first build)
